# @codepop/macos

CodePop macOS 原生应用，基于 Electron 构建。

## 功能

- macOS 原生界面
- 菜单栏快捷访问
- 本地代码索引
- 与 CLI 协同工作

## 安装

```bash
pnpm install
```

## 开发

```bash
# 开发模式
pnpm dev

# 构建 macOS 版本
pnpm build:mac
```

## 系统要求

- macOS 11.0+
- Apple Silicon 或 Intel 处理器

## 技术栈

- Electron 28
- React 18
- electron-builder
